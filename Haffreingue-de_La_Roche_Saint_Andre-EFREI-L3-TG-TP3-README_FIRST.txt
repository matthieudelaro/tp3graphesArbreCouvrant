 = = = = README FIRST  = = = =

Ce programme a été testé sur avec Mac OS X Mavericks.

Vous pouvez lancer le programme en ligne de commande simplement :
$ ./tp3graphes
Which file do you want to open ? Haffreingue-de_La_Roche_Saint_Andre-EFREI-L3-TG-TP3-5.txt

Vous pouvez aussi spécifier directement le nom du fichier que vous souhaitez ouvrir.
$ ./tp3graphes Haffreingue-de_La_Roche_Saint_Andre-EFREI-L3-TG-TP3-5.txt

Nous avons retenu la syntaxe de fichier présentée dans le sujet du TP.
